## 01 - seaborn-data 全数据集探索

对 seaborn-data 仓库中所有 22 个数据集逐一分析，图表采用 Nature 期刊风格。

涉及的分析方法：
- 分组统计、相关性分析、时间序列趋势
- KMeans 聚类、线性回归、Sigmoid 曲线拟合
- Bootstrap 置信区间、KDE 密度估计
- 层次聚类、雷达图、热力图、堆叠面积图

数据集覆盖：Titanic、Penguins、Iris、Diamonds、MPG、Flights、Anscombe、Geyser、Dowjones、Healthexp、Seaice、Planets、Car_crashes、Taxis、fMRI、Exercise、Attention、Anagrams、Dots、Glue、Brain_networks + 彩蛋

## 环境

- Python 3.12
- Jupyter Notebook（VSCode）
- 主要依赖：pandas, numpy, matplotlib, seaborn, scikit-learn, scipy
